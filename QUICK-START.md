## 📞 Need Help?

**Common Issues:**

**"Netlify upload failed"**
→ Make sure folder contains `landing-page.html` at root level

**"Gumroad won't publish"**
→ Complete seller verification (requires ID, takes 5-10 min)

**"Questionnaire link not working"**
→ Make sure form is published (not in draft mode)

**"Payment not processing"**
→ Check Gumroad account is verified and payout info is complete
